# Backend

Backend nay duoc tach rieng khoi frontend va chay doc lap bang Node.js core.

## Chay local

```bash
cd Backend
node server.js
```

Mac dinh backend chay tai `http://localhost:4000`.

## Tai khoan demo

- Owner: `owner@example.com` / `123456`
- Staff: `staff@example.com` / `123456`

## Ghi chu

- Du lieu duoc luu o `Backend/data/store.json`
- Backend hien tai dung JSON store de demo flow thay cho DB that su
- Muc tieu la tach frontend va backend rieng biet de sau nay thay bang PostgreSQL / Redis / WebSocket de dang hon
